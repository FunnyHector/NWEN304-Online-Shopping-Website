Note: Our add POST services require a file buffer when posting. Not quite sure how to post an image so I didn't make a curl for it.
==============================================================================================

Service: '/add/vinyl'
Method: POST
Description: Used to add a new album to the database.
Required:
- Title
- Artist name
- Valid album release date
- Valid price
- Image for album cover
Success: If the information is valid, a message will be displayed notifying the user that the album was added successfully. The information is also updated in the database.
Fail: If the values are invalid, an error will be rendered telling the user to provide valid information. 

----------------------------------------------------------------------------------------------
Service: '/add/vinyl'
Method: GET
Description: Provides a page with a form for adding a new album.
=> curl -X GET https://vinylholics.herokuapp.com/add/vinyl
